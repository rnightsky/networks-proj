About This Project

This file sharing system was developed as part of the CSC 430 Computer Networks course. It features:

  • Secure file uploads and downloads
  • File integrity verification using SHA-256 hashing
  • Automatic file versioning
  • Responsive web interface
  • Detailed logging of all operations

The system is designed to demonstrate the principles of client-server architecture, network protocols, and secure file transfer.
